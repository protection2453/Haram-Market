# Haram-Market

## Introduce
- 본 프로젝트는 폐사업장에서 나오는 버려질 물건들을 신규 사업자에게 판매하는 쇼핑몰입니다.
- 기획 1명, 개발 1명, 기획/개발 보조 1명으로 진행한 프로젝트입니다.
- Wordpress와 plugin을 활용하여 많은 기능을 짧은 프로젝트 기간동안 제작했습니다.

## Environment
- OS : Redhat
- Editor : Visual Studio Code
- Language : Html5, css, javascript, php
- DB : MariaDB
